About Burchfield.info

Syntax & Tech

Cent OS 7
Using Digital Ocean, I was able to create a virtual machine that lives, "In the Cloud." This allows for more consistant uptime, as well as cheaper electric bills. Couple that with the chance to skip out on purchasing costly server hardware, and its no wonder developers are making the switch to services such as AWS; Digital Ocean, Azure, Cloud Services, Rackspace, the list goes on.
Apache

Using Apache Webserver 2, Burchfield.info is able to serve up pages on a minimal operating system that is hardened for security.

Certbot

Using Certbot, Burchfield.info is able to securley communicate with users. Https, or Hyper Text Transfer Protocol Secure, is a method of encrypting traffic before being sent across the internet to its destination, i.e., your web-browser!

HTML5
Burchfield.info uses HTML5 to structure and frame its content. Given its easabiltiy, this language is one of the first stepping stones on the lengthy journey to become a webmaster.
CSS3
Cascading Style Sheets help make the formatting of content possible. That said, most Development is switching to frameworks such as Bootstrap.JS; Pure.css, Material-UI, Responsive, this list just never ends. However, modifying the attributes of these frameworks requires knowledge of CSS.
Bootstrap.JS
What is Bootstrap?
Bootstrap is a CSS framework designed to save time, while enhancing the functionality and aethestics of HTML5 code. This framework is almost soley responsible for this page and its font-family, font-sizing, page colors, etc.
